# MovieRentalV2

Do stworzenia projektu, korzystaliśmy z:
- PHP 8.0.10
- Symfony 6.0.1
- PHPUnit 9.5.10
- MySql 8.0.21

Do nazewnictwa klas stosowaliśmy PascalCase, natomiast do nazywania funkcji i zmiennych camelCase.
